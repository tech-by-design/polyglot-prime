This data grid provides a summary overview of incomplete file groups that resulted in validation errors, displaying key information such as Created Time, Tenant ID, Interaction ID, URI, User Agent, and CSV Zip File Name. The grid highlights submissions where mandatory files for validation are missing or incomplete, enabling users to quickly identify problematic file groups. A drill-down feature enables users to explore detailed insights for each entry by clicking the expand arrow next to any row. This reveals specific validation errors including Error Type, Error Message, and Description for each incomplete file within the group, providing comprehensive context for efficient error resolution.
