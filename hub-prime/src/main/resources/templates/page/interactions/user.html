This widget displays user session activity within the TechBD platform. It includes Session Start Time, User ID, and Username, providing a consolidated view of user login sessions and activity. A drill-down feature is available on the Session Start Time column. Clicking a session displays detailed activity recorded during that session, including Recorded Time, TechBD Interaction ID, URI Accessed, User Role, IP Address, and User Agent. This enables administrators to audit user actions, monitor system usage, and investigate session activity.
